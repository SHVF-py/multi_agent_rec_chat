Embed Quiribot on Your Site

Copy and paste one script tag — no coding required.

Step 1

Your Site Key

This unique key identifies your store. Never share it publicly in plain text outside of the script tag.

{{ business.site_key }}
Step 2

Copy the snippet

Add this tag just before the closing </body> tag on every page of your site.

<!-- Quiribot AI Shopping Assistant -->
<script
  src="http://localhost:7000/widget/loader.js"
  data-site-key="{{ business.site_key }}"
  data-api="http://localhost:8080">
</script>
Step 3 — Platform-specific instructions

Where to paste it

{% if business.platform == 'woocommerce' %}

WooCommerce / WordPress: Go to Appearance → Theme Editor → footer.php and paste the snippet before </body>. Or use a plugin like Insert Headers and Footers.

{% elif business.platform == 'shopify' %}

Shopify: Go to Online Store → Themes → Edit Code. Open Layout/theme.liquid and paste before {% endbody %}.

{% else %}

Paste the script tag before the closing </body> tag in your HTML template.

{% endif %}
Step 4

Test it

Open your store in a new browser tab. You should see the Quiribot chat button in the corner within a few seconds. Click it to start a conversation!
